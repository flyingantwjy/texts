测试vscode安装有没有问题：点 File (最左上角），然后 点 第一个 New，然后取个名字，随便啥名都可以，类型设置为python：点 File ---save as---类型改为Python
创建python文件：创建一个python文件的方法： File---new text file----取名，改类型txt为python
输入python代码：代码都要顶着左边写。顶着左边输入，不要空格，括号必须在英文输入法状态下输入，引号也是。电脑的右下角可以看出来是中文还是英文输入，你可以按 ctr 和 shift切换输入法
运行python代码：点vscode右上角的运行按钮
结果显示区:TERMINAL或者终端
如何创建 Python文件：File---New Text File---File----Save as---改名和类型
vscode界面里除了 .py的 其它都可以叉掉
注释掉：输入新的代码时，刚才的代码如果舍不得删掉可以这样处理：代码前加#，这叫注释掉。相当于给代码贴了封条，暂时不用，以后想用的时候把#去掉就可以了。注释掉一行代码 前面加 #就可以了。如果好几行需要注释掉，需要这样操作： 鼠标选中要注释掉的几行内容，然后点 编辑（软件左上角第二个）点 倒数第二个 块注释
python代码里的标点：python代码里的标点都必须是半角符号，即在英文输入法状态下输出的
如何看python运行错误：首先看最后一行，syntaxerror这个错误一般都是大小写错误，标点符号错误，缩进错误等；其次看line ...,这里告诉我们第几行出错了
引号：如果输出的内容本身有 ' 这个，那么外面必须用双引号，不能用单引号；如果左右是三引号，代码里是什么样，结果也会是什么样，这个叫所见即所得，我们可以用这个功能来画画
字符串：不管是单引号，双引号还是三引号，这样的东西，我们称之为字符串。左右有引号的,加引号（双 单 三）都是。引号加文字是字符串，必须左右都有引号，左右的引号必须是一样的。'hello world'是字符串，“I'm good"是字符串，''' This is my friend''' 是字符串
空格：= 前后不加空格也行，但是一般为了美观，都会加上
空行：代码之间的空行，有和没有都没有区别，加空行也只是为了美观，或者看起来整洁一点
变量：= 左边的就是变量，=右边的就是这个变量的值。范例：name = 'kaori' 可以这么解释：有一个变量name，它的值是字符串'kaori'；num = 12 这行可以这么说：变量num的值是12
数据类型，数据类型包括：字符串，整数，浮点数。12345是整数。浮点数就是我们普通人理解的小数
变量名：随便写，但一般最好是英文字母或英语单词，也可以英语字母+数字，例如a1；英语单词+数字 如  list2。但要注意有些英语单词，在Python里有特殊用处，我们尽量避免。比如 list是列表，用了在运行时有时会出错，所以 一般会改成 list1 这样
输出，输出必须要用 print( )
函数：英语单词（）这样的 我们叫它 函数。函数就是起特定作用的代码。具体作用一般通过函数的英语单词的意思可以看出来
+的用法：1. 加法符号，如 print(3+4)；2. 字符串的拼接符  意思是 它用来将两个字符串拼在一起。第二种用法 在写代码时经常会出错，一定要检查 + 的左右必须同时字符串，如果 有一个是整数或浮点数 就会出error。+前后空格和不空格都可以
input函数：input函数的作用是让用户输入。
主谓一致：主语是单数，谓语+s；主语是复数，谓语不加s。例题：The book____(looks/look) nice. 答案是：looks，不是look
